# Sistema Nóminas
Una aplicación que consiste en un sistema de nóminas para una empresa. La aplicación permite ver los empleados, agregar empleados(fijos o eventuales)...

## Versiones
V1.0 - ArrayList (Listas) 
V2.0 - TreeSet (Árboles)
V3.0 - HashMap (Mapas)
